Java Examples

Example 1: Hello World

This program prints "Hello, World!" to the console.

                    
                        public class HelloWorld {
                            public static void main(String[] args) {
                                System.out.println("Hello, World!");
                            }
                        }

Output:

                        
                            Hello, World!

Example 2: Variables and Arithmetic

This program demonstrates the use of variables and performs simple arithmetic operations.

                    
                        public class ArithmeticExample {
                            public static void main(String[] args) {
                                int a = 5;
                                int b = 3;
                                int sum = a + b;
                                int product = a * b;
                                System.out.println("Sum: " + sum);
                                System.out.println("Product: " + product);
                            }
                        }

Output:

                        
                            Sum: 8
                            Product: 15

Example 3: Conditional Statements

This program uses if-else statements to determine if a number is positive, negative, or zero.

                    
                        public class ConditionalExample {
                            public static void main(String[] args) {
                                int number = -5;

                                if (number > 0) {
                                    System.out.println("The number is positive.");
                                } else if (number < 0) {
                                    System.out.println("The number is negative.");
                                } else {
                                    System.out.println("The number is zero.");
                                }
                            }
                        }

Output:

                        
                            The number is negative.

Example 4: Looping

This program uses a for loop to print numbers from 1 to 5.

                    
                        public class LoopExample {
                            public static void main(String[] args) {
                                for (int i = 1; i <= 5; i++) {
                                    System.out.println(i);
                                }
                            }
                        }

Output:

                        
                            1
                            2
                            3
                            4
                            5

Example 5: Functions

This program defines a simple function and calls it from the main method.

                    
                        public class FunctionExample {
                            // Function to add two numbers
                            static int add(int a, int b) {
                                return a + b;
                            }

                            public static void main(String[] args) {
                                int result = add(3, 4);
                                System.out.println("Sum: " + result);
                            }
                        }

Output:

                        
                            Sum: 7
